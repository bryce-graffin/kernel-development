## Kernel Modules
Pieces of code that can be loaded/unloaded into a running kernel without recompiling the
whole thing. Great sandboxed way to get a feel for kernel programming.
- good tutorials available at these sites:
	- The Linux Documentation Project ([tldp.org](https://www.tldp.org/))
	- [kernel.org](https://www.kernel.org/)

## Virtual Machine
Likely will be written/tested inside of a VMWare Fusion virtual machine. I have used this in
the past to run an ArchLinux ISO on my M4 MacBook Air.

This is *strongly advised* so there is not a risk of bricking my actual system while learning.
